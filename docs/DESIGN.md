# 设计系统定义

## 2.1 色彩体系

### 品牌色（Primary）

| 名称 | 色值 | 说明 |
|---|---|---|
| 主色 | `#8B5CF6` | 柔紫，跨性别旗帜中紫色的柔和版本 |
| 主色浅 | `#C4B5FD` | 薰衣草 |
| 主色极浅 | `#EDE9FE` | 紫雾 |
| 主色深 | `#6D28D9` | 深紫 |

### 辅助色（Secondary）

| 名称 | 色值 | 说明 |
|---|---|---|
| 暖粉 | `#F9A8D4` | 玫瑰粉 |
| 暖粉浅 | `#FDF2F8` | 粉雾 |
| 天蓝 | `#93C5FD` | 天蓝，跨性别旗帜蓝色的柔和版 |
| 天蓝浅 | `#EFF6FF` | 蓝雾 |

### 功能色

| 类型 | 主色 | 背景 |
|---|---|---|
| 成功 | `#34D399` | `#ECFDF5` |

| 错误/价格 | `#F87171` | `#FEF2F2` |
| 信息 | `#60A5FA` | `#EFF6FF` |

### 中性色

| 名称 | 色值 | 说明 |
|---|---|---|
| 文字主 | `#1E1B4B` | 墨紫，比纯黑更温柔 |
| 文字副 | `#6B7280` | 次级正文 |
| 文字辅 | `#9CA3AF` | 辅助文字 |
| 边框 | `#E5E7EB` | 通用边框 |
| 背景 | `#FAF5FF` | 极淡紫，替代冷灰 `#fafafa` |
| 卡片 | `#FFFFFF` | 白色卡片背景 |
| 分割 | `#F3E8FF` | 淡紫分割线 |

### 配色理念

跨性别旗帜的三色（蓝、粉、白）作为灵感源头，但通过降低饱和度和提升明度使其变为柔和的薰衣草紫、玫瑰粉与天蓝组合。紫色作为主色调兼具“神秘感”（契合“魔女”品牌名）和包容性。整体气质应像走进一家开满薰衣草的温馨小药房。

## 2.2 字体系统

- 字体栈：`"LXGW WenKai", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 说明：首选霞鹜文楷，增添手写温暖感；系统字体作为 fallback

### 字号阶梯

| 类型 | 字号 | 字重 | 字距 | 行高 |
|---|---:|---:|---|---:|
| Display | 32px | 800 | -0.02em | 1.2 |
| H1 | 26px | 700 | 默认 | 1.3 |
| H2 | 20px | 600 | 默认 | 1.4 |
| Body | 15px | 400 | 默认 | 1.75 |
| Caption | 13px | 400 | 默认 | 1.6 |
| Badge | 12px | 600 | 默认 | 1 |

## 2.3 间距与圆角

### 间距

- 间距基础单位：`4px`
- 常用间距：`8 / 12 / 16 / 20 / 24 / 32 / 48`

### 圆角

| 场景 | 圆角 |
|---|---|
| 卡片/面板 | `20px` |
| 按钮 | `12px`（主按钮） / `999px`（胶囊按钮） |
| 输入框 | `12px` |
| 徽章/标签 | `999px` |
| 弹窗 | `24px` |
| 图片 | `14px` |

## 2.4 阴影

| 名称 | 值 | 用途 |
|---|---|---|
| `shadow-sm` | `0 2px 8px rgba(139, 92, 246, 0.04)` | 卡片基础阴影 |
| `shadow-md` | `0 8px 24px rgba(139, 92, 246, 0.06)` | hover 或浮层 |
| `shadow-lg` | `0 20px 40px rgba(139, 92, 246, 0.08)` | 弹窗、重点容器 |
| `shadow-glow` | `0 0 0 3px rgba(139, 92, 246, 0.15)` | 焦点状态 |

## 2.5 组件规范

### 按钮

#### 主按钮（Primary）

- 背景：线性渐变 `#8B5CF6` -> `#7C3AED`
- 文字：`#FFFFFF`
- 圆角：`12px`
- Padding：`14px 24px`
- Hover：渐变加深 + `shadow-md`
- Disabled：`opacity: 0.5`

#### 次按钮（Secondary）

- 背景：`#EDE9FE`
- 文字：`#6D28D9`
- Hover：`#DDD6FE`

#### 购物车按钮

- 背景：线性渐变 `#F9A8D4` -> `#F472B6`
- 文字：`#FFFFFF`

#### 危险按钮

- 背景：`#FEE2E2`
- 文字：`#DC2626`

### 卡片

#### 商品卡片

- 背景：`#FFFFFF`
- 边框：`1px solid #F3E8FF`
- 圆角：`20px`
- 阴影：`shadow-sm`
- Hover：`translateY(-4px)` + `shadow-md`
- 顶部装饰条：`4px` 高，颜色取自分类 `accent`，默认 `#C4B5FD`

### 输入框

- 背景：`#FFFFFF`
- 边框：`1px solid #E5E7EB`
- 圆角：`12px`
- Padding：`12px 16px`
- Focus：边框 `#8B5CF6` + `shadow-glow`
- Placeholder：`#9CA3AF`

### 徽章/标签

#### 订单状态

| 状态 | 背景 | 文字 |
|---|---|---|
| 待处理 | `#FDF2F8` | `#BE185D` |
| 已发货 | `#EFF6FF` | `#1D4ED8` |
| 已完成 | `#ECFDF5` | `#059669` |
| 已取消 | `#F1F5F9` | `#64748B` |

#### 购物车角标

- 背景：`#F472B6`
- 文字：`#FFFFFF`
- 说明：使用粉色而非纯红，保持整体视觉更柔和

## 2.6 品牌元素

| 元素 | 定义 |
|---|---|
| Logo 文字 | `魔女小店`，使用 `LXGW WenKai` 字体 |
| Logo 装饰 | 文字前带一个小魔法棒 `✨` 或小药瓶 `🧪` emoji |
| Slogan | `做最真实的自己`，用于首页 hero 副标题 |
| Favicon | 保持 SVG，考虑改为紫色系小魔法帽或药瓶轮廓 |
