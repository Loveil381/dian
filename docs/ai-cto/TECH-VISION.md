## 已达成的创新点

- 零框架原生 PHP 商城，适合廉价虚拟主机
- 安装向导一体化完成“建表 + 配置 + 建管理员”
- 邮件发送采用 SMTP socket 优先、`mail()` 回退的双模式方案
- 统一日志工具 `shop_log()`，默认写入 `logs/app.log`，失败时回退 `error_log()`
- 前台事件委托统一到 `data-*` 属性，降低模板与脚本耦合

## Round 19-20 安全加固记录

- 安装器增加 BOM 防护与 prefix 输入校验
- 安装入口统一为 `install.php`，`admin/setup.php` 仅保留重定向
- checkout 创建订单状态修正为 `pending`
- 目录封禁扩展到 `database/` 和 `logs/`
- 后台登录新增 5 次失败锁定 15 分钟机制，并记录警告日志
- 未知路由统一返回 HTTP 404 友好错误页

## v1.5.0 技术亮点

- **纯 CSS 柱状图**：dashboard 7 天趋势图用 flexbox + PHP 计算高度百分比实现，零 JS 依赖
- **操作日志基础设施**：`shop_admin_log()` 函数 + try-catch 静默降级，表不存在时不影响主流程
- **优惠券原子操作**：`UPDATE SET used_count = used_count + 1 WHERE used_count < usage_limit` 防并发超限
- **事件驱动通知**：`shop_notify_order_event()` 统一入口，各通知独立开关，失败不阻断主流程
- **速率限制扩展**：优惠券码输入加 `shop_rate_limit()` 防暴力枚举
